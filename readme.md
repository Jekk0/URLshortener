#URL shortener

##Installation and starting up the project

1. Copy the project to your server:
	git clone https://github.com/Jekk0/URLshortener.git

2. In the project folder do:
	* php composer install;
	* php artisan key:generate;
	* php artisan migrate; 

3. Add to .ENV file:
	* Database name
	* User name 
	* Password
